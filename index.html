Features

Trully Decentralised

ZIFF is a decentralized mesh network that does not rely on any central infrastructure.

Complete Security

ZIFF uses protocol-enforced end-to-end encryption to protect user data and privacy.

Full Control

ZIFF offers a peer-to-peer network access. You can forget about paying for internet access.

Our Mission

We want to return to the internet's original promise: truly private, free, and democratic.

While current solutions like VPNs and proxy servers offer a layer of protection, they rely on centralized servers that remain vulnerable to attack.

ZIFF takes a fundamentally different approach.

Our first global mesh network will let you connect to the internet from anywhere for free. Through a secure, decentralized infrastructure, ZIFF ensures that your internet will be stable and secure from shutdowns, and your data is safe from malicious actors and trackers.

Roadmap

Aug 2024 - Oct 2024

Development of basic algorithms and data structures for the main processes of the network activity (routing and addressing).

Nov 2024

Testing of basic algorithms and data structures.

Dev 2024 - Feb 2025

Configuring the virtual network drivers of the system core, adding multicasting.

Mar 2025

Testing of the VND and multicasting.

Apr 2025 - Jul 2025

Adding all low-level services to a single API. Creating a single core by loading all the API functionality into it. Creating an admin panel as a CTL library also using all API functionality.

Aug 2025 - Sep 2025

Testing of API, single core and CTL library. Release alpha version of ZIFF.
